# ecommerce-platform

一个电商项目工作区，包含后端、前端、测试以及数据库设计文档。

## 目录结构

- `ecommerce-backend/`：Node.js / Express 后端
- `ecommerce-frontend/`：前端示例页面
- `ecommerce-tests/`：测试用例
- `ecommerce_database_design.md`：数据库设计文档
- `ecommerce_database_setup.sql`：数据库初始化脚本
- `database_design_summary.md`：数据库设计总结
- `database_performance_guide.md`：数据库性能建议
- `project_summary.md`：项目总结

## 快速开始

### 后端
```bash
cd ecommerce-backend
npm install
npm run dev
```

或：
```bash
cd ecommerce-backend
node src/server.js
```

### 前端
直接打开：
- `ecommerce-frontend/index.html`

### 测试
测试文件位于：
- `ecommerce-tests/`

当前仓库用于集中整理电商相关代码与文档。
